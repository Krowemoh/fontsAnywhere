What is Fonts Anywhere?

Fonts Anywhere is a bookmarklet to check out all the fonts in Google Fonts. Instead of dealing with html, css and refreshing, you can sit back, relax and spam the up and down arrow keys while fonts change all around you.

The arrow keys now working?

Make sure the search bar has focus, if it does it will be glowing blue.
